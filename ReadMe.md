\# Tyrant

\#\# Setup

\#\#\# Requirements  
\- Unity version (e.g., 2022.3.50f1)  
\- Platform (Windows / Mac / Linux)

\#\#\# How to Run  
1\. Clone the repository  
2\. Open the project in Unity  
3\. Load the main scene  
4\. Press Play or open the Build folder

\#\# Features  
\-Third-person over-shoulder camera  
\-Basic gun play  
\-Basic zombie AI behavior  
\-Basic inventory  
\-Pickable items  
\-Door opening and auto-closing  
\-Weapon and door sounds 

\#\# Controls  
\- WASD – Move  
\- Mouse – Camera  
\- Left Click  – Shoot  
\- Right Click  –  Aim  
\- F – Interact / Pickup  
\- TAB  – Inventory
